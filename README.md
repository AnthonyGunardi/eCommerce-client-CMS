# ecommerce-client-cms
"AnTz Shop" e-commerce CMS app, created using Vue.js, bootstrap (CMS client & customer client) & node.js, express, sequelize, postgreSQL (server)


## URL
https://ecommerce-cms-antzshop.web.app
